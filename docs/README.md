# WiseTrader: Stock Prediction & Analysis Platform

WiseTrader is an application designed to provide insights into stock price movements using various machine learning and deep learning models. It offers a Streamlit-based interactive interface for users to explore stock data and predictions powered by models like Convolutional Neural Networks (CNN) and Long Short-Term Memory (LSTM).

## Features

*   **Interactive Stock Watcher**: Monitor real-time (1-minute interval) stock data for any ticker.
*   **Multiple Prediction Models**: 
    *   **Wise CNN**: Utilizes Convolutional Neural Networks for binary classification (predicting if the price will go up or down).
    *   **Wise LSTM**: Employs Long Short-Term Memory networks for time-series forecasting of stock prices.
    *   **(Under the Hood)** **Logistic Regression**: A classical machine learning model for binary classification of stock movements.
*   **Technical Feature Engineering**: Tools to generate various technical indicators (e.g., SMA, EMA, RSI, MACD) to enrich stock datasets for model training.
